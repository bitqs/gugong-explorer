# 故宫漫游 · 2.5D 网页探索游戏 — 设计文档

日期：2026-06-22
状态：已批准，待实现

## 一句话

浏览器里操控一个小人在手绘故宫图上行走，走近或点击宫殿弹出名称与介绍。保留全部手绘美术。

## 背景与原则

- 现有资产：两张瓦片拼接的手绘故宫鸟瞰图（轴测投影）。
  - `gugong-full-map-z5.png` — 4608×7680，47MB
  - `gugong-full-map-z6-max.png` — 8704×15104，159MB
- 核心判断：图是**平面插画**（轴测投影伪 3D），不含深度数据。其价值在手绘美术本身。
- 因此不做真 3D 重建（会丢掉美术、工作量以月计），而做 **2.5D 探索**：直接用这张图当世界，轴测投影天然读起来就有立体感。
- 平台：网页（一个链接即可分享，手机/电脑都能开）。
- 玩法：v1 = 自由漫游 + 点击介绍。最小可玩核心，后续增量加。

## v1 设计

### 1. 美术与性能

原图过大，浏览器无法直接加载。

- v1：把 `z5` 图降采样到约 **3072px 高**的单张 PNG 作为背景（`assets/map.png`）。依然清晰，单次加载、内存约 20–30MB，手机可承受。
- 深度缩放瓦片金字塔（无限放大看细节）留作 v2。

### 2. 玩法与操控

- 移动：方向键 / WASD（桌面）+ 手机虚拟摇杆（触屏）。
- 镜头跟随小人；画面边缘视差/暗角营造纵深。
- v1 **自由行走**，无逐建筑碰撞，仅世界边界限制。碰撞遮罩留 v2。
- 缩放：滚轮 / 双指，限定一个合理区间（避免糊或越界）。

### 角色美术

- 风格定为**匹配地图的手绘/平涂工笔风**（不用像素风）。
- 因此 **不使用 PixelLab**（它是像素风生成器，与工笔画冲突）。PixelLab 仅在未来想加像素彩蛋时才考虑。
- v1 实现：用一个小的平涂风格小人，调色板贴近地图主色（朱红/金黄/青灰）。来源可为手绘 PNG，或先用 canvas/SVG 画一个简洁平涂小人顶上，后续替换为更精致版本。

### 3. 热点（核心内容）

- 沿中轴线手工标注约 **12 座关键建筑**：午门、太和殿、中和殿、保和殿、乾清宫、交泰殿、坤宁宫、御花园等。
- 每座记录 `{ id, 名称, 像素区域(矩形或多边形), 一两句介绍 }`。
- 交互：小人走近 → 高亮该热点；点击（或走入区域）→ 弹出介绍卡。
- 数据存 `assets/pois.json`，新增建筑只改数据、不改代码。

### 4. 技术栈

- 纯静态页 + **PixiJS**（CDN 引入，无构建步骤）。
- `index.html` 直接打开即玩；部署只需丢到任意静态托管。
- 镜头 / 缩放 / sprite 用 Pixi 容器变换处理。
- 备选：纯 vanilla Canvas（零依赖但需自写镜头数学）。已选 Pixi。

### 5. 文件结构

```
index.html         — 入口，引入 PixiJS CDN
src/game.js        — 主循环、镜头跟随、输入（键盘 + 触屏摇杆）
src/pois.js        — 加载热点、走近高亮、弹卡 UI
assets/map.png     — 降采样背景（约 3072px 高）
assets/pois.json   — 建筑数据（id/名称/区域/介绍）
```

### 组件边界

- **game.js**：拥有渲染循环、世界/镜头变换、输入。对外提供「当前小人世界坐标」。不知道热点的存在。
- **pois.js**：读 `pois.json`，监听小人坐标，负责高亮与弹卡 DOM。依赖 game.js 暴露的坐标与坐标变换。
- **pois.json**：纯数据，零逻辑。

### 数据流

输入（键盘/摇杆）→ game.js 更新小人世界坐标 → 镜头跟随重算变换 → 渲染。
每帧 pois.js 用小人坐标做最近热点判定 → 高亮 / 允许点击 → 弹卡。

### 错误处理

- 资产加载失败（map.png / pois.json）：页面显示明确错误提示，而非空白。
- `pois.json` 缺字段：跳过该条并在 console 警告，不崩溃。

### 测试与验收

v1 完成的判定：
1. 浏览器打开 `index.html`，地图显示、清晰。
2. WASD/方向键能移动小人，镜头跟随，不越界。
3. 走近任一标注建筑触发高亮；点击弹出正确的名称+介绍卡。
4. 手机上触屏摇杆可移动、可点击弹卡。
5. 至少 12 座中轴线建筑有数据。

## 明确不做（v1，留 v2）

- 逐建筑碰撞（碰撞遮罩）
- 深度缩放瓦片金字塔（无限放大看细节）
- 遮挡（走到殿后被建筑挡住的 z-ordering）
- 任务 / 集章 / 解谜
- 音效 / 音乐
